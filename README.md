# CVE Mining Project

### Requirements
  * Scrapy 2.4.1
  * Python 3.8.8
  * Pandas 1.2.3
### Description
A semi-automated python program to automated the process of obtaining CVE information from MITRE's CVE database.
https://cve.mitre.org

#### Usage
Input: 
- List of CVE IDs

Output: 
- CVE Link, CVE Description, and Reference Links in the format of a CSV file
